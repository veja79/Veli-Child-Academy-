# Veli Child Academy

Official website and administration portal for Veli Child Academy.

# Project Overview

This project is being developed to provide an online presence for Veli Child Academy while also offering an administration portal for managing school announcements and news.

# Features

# Public Website
- Home
- About the School
- Announcements
- Contact Us

# Administration
- Secure Login
- Manage Announcements
- Manage News

# Technology Stack

# Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA

# Frontend
- React
- TypeScript

# Database
- MySQL

 Currently in Development
